Fichero README - Algoritmos de busqueda A*
---------------------------------------
Ejemplos implementados en Python de el algoritmo de busqueda A* sobre espacios de busqueda en forma de arboles

Se da un ejemplo sobre un arbol binario de 4x4 y otro 3x3.
La version de Python utililtzada es la 3.9

Se puede compilar tanto en PyCharm como en google colabs(https://colab.research.google.com/drive/1CFVK3fsjcR6ovICfyEQBbXbpfjte2_4Y?usp=sharing).

Se ofrecen dos tipos de ficheros .py para PyCharm i otro .ipynb para google colabs.

Para ejecutar en google colabs, se selecciona la pestaÃ±a Runtime i Run All.


ANACONDA
========
Si se tiene el entorno ANACONDA instalado, pasos a seguir:

1. Abrir ANACONDA

2. Abrir (Launch) el Spyder (sino esta instalado, instalarlo previamente: desde el boton Install del propio spider).

3. Una vez en el entorno que ofrece Spyder, podeis observar que se abre en forma de dos ventanas. 

3.0 En caso de duda, para consultar la version de Python: 
python --> About spider
Ayuda --> Acerca de Spyder..

3.1. Abrir uno de los ficheros . py (con la version instalada)
File --> Open
Se os cargara en la ventana de la izquierda

3.2 Clicar el triangulo verde ( que corresponde a "Run file (F5)").
Se os ejecutara en la ventana de la derecha.
